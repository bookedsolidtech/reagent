Add business advisory, design, content, and specialist engineering agents

New agent categories:
- **business/**: `ceo-advisory` — fractional CEO for vision, OKRs, revenue model, fundraising
- **design/**: `figma-design-specialist`, `figma-code-connect`, `figma-component-architect` — Figma Variables/tokens, Code Connect CLI mapping from CEM, and programmatic Figma library generation from code

- **marketing/**: `marketing-advisor`, `social-media-advisor` — CMO-level brand/GTM and multi-platform social strategy
- **legal/**: `legal-advisor`, `tax-advisor` — business legal frameworks and tax strategy

New engineering specialists:
- `integration-test-engineer` — hermetic integration tests for CLI/daemon/state machine boundaries
- `cli-dx-engineer` — TUI prompts, `isTTY` detection, shell completion, interactive init wizards
- `release-engineer` — Changesets discipline, semver governance, npm exports correctness, provenance
- `web-components-standards-engineer` — CE v1 spec, CEM tooling, form-associated elements, ARIA in shadow DOM, framework interop
- `design-token-engineer` — Style Dictionary pipeline, CSS/JS/Tailwind token output, CI validation
